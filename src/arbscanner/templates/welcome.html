Getting started

1

Access the Pro dashboard

Open the live dashboard and add your key to requests using the X-Api-Key header, or browse directly — the dashboard auto-detects it.

2

Set up alerts

Add TELEGRAM_BOT_TOKEN and TELEGRAM_CHAT_ID (or DISCORD_WEBHOOK_URL) to your instance's .env to receive push alerts when edge exceeds your threshold.

3

Authenticate API calls

Pass your key as X-Api-Key: {{ api_key or "your-key-here" }} on any /api/* request to unlock real-time data and calibration context.

4

Run the scanner locally (optional)

Clone the repo, fill in .env, and run arbscanner scan for a full terminal dashboard with live execution support.
