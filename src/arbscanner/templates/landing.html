Why ArbScanner?

Nobody else monitors all overlapping markets continuously with net-of-fees calculation and calibration context.

Automated Market Matching

AI-powered matching of identical markets across platforms using semantic embeddings and LLM confirmation. "Will the Fed cut rates?" matches "KXFEDCUT-26JUN" automatically.

Real-Time Scanning

Continuous monitoring of all matched pairs with 30-second refresh cycles. Order book data, not just mid-prices.

Exact Kalshi bracket-based fee schedule and Polymarket taker fees. See actual edge, not theoretical spreads.

How It Works

Fetch Markets

Pull all active binary markets from Polymarket and Kalshi via the pmxt unified API.

Match Markets

Sentence-transformer embeddings + Claude LLM confirmation identify the same event across platforms.

Scan for Arbs

For each matched pair, pull order books and check if buying YES on one + NO on the other costs less than $1.00.

Score & Alert

Subtract fees, rank by expected profit, add calibration context, and alert you when real edge appears.
